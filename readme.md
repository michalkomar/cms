How to run project
==================

1. CLI: `composer create-project wundermanpraha/cms`
1. CLI: `make setup`
1. Change properties in PROJECT_ROOT/app/config/config.local.neon to your servers
1. Go to root of project and run in CLI: `composer install -o --no-dev`

Dependencies
============

+ PHP >= 5.6
+ MySQL
+ Composer
+ Nginx | Apache 2
